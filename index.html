5-stage Learning Path
  • 17 SQL queries organised by workflow stage: Define Objectives, Design Study, Collect Data, Analyse and Infer, Statistical Modelling
  • Each stage retrieves terms relevant to that phase of the research workflow
  • Later stages (D and E) incorporate formula, variable, value range, and usage condition fields, reflecting increased mathematical formalisation
  • Enables progressive disclosure: users first encounter conceptual vocabulary, then formal mathematical descriptions
COBUILD's Way
  • Corpus-first access inspired by Sinclair's corpus lexicography
  • Q17 (COBUILD Parametric Search): simultaneous wildcard matching across English name, Chinese name, example sentences, and collocations
  • Users discover entries from usage context rather than searching by term name
  • Dual presentation modes: Formal Definition (formula-first) vs. COBUILD-style (usage-first), serving both expert researchers and learners
Linguistic Features
  • Etymology root search (Q10): accepts Latin/Greek roots and eponymous names (Fisher, Pearson, Cohen, Bayes, Galton)
  • Confusable term discrimination (Q13-Q14): retrieves easily confused pairs (accuracy vs. precision, fixed effect vs. random effect, variance vs. standard deviation)
  • Full-text keyword search (Q15): wildcard across Chinese name, English name, Chinese definition, and related concepts
  • Supports contrastive terminological analysis for learners and translators
Specialized Queries
  • Symbol reverse-lookup (Q12): users enter mathematical symbols (σ, μ, χ, β, κ) and retrieve corresponding terms via wildcard matching on formula and variable fields
  • Browse by subcategory (Q6) and discipline (Q7): wildcard search across statistical subcategories and cross-disciplinary fields (linguistics, psychology, ecology, economics, medicine, engineering)
  • Cross-disciplinary aggregation (Q8): groups terms by discipline and pipeline stage, identifying the most cross-disciplinary concepts
  • Terms with formulas (Q11): retrieves all entries with non-null formula field for technical reference
📄 Termbank Animations/index.html
Integrated Visualisation Resources
  • Portal index page (index.html) listing all 40 terms with associated demonstration resources, serving as the central navigation interface
  • 4 standalone HTML animations, each designed around a linguistic research scenario and fully offline-capable: Linear Mixed-effects Model (fixed vs. random effects and shrinkage), Cohen's Kappa (contingency table construction), Bootstrap (resampling methods), Interaction Effect (factorial designs)
  • Demonstration resource field stores direct references to HTML files with applicability scoring (1-3 stars)
  • Supplementary query (Supp_Q1) retrieves entries with visualisation resources ranked by applicability score
